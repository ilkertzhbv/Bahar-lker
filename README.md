# Bahar ❤️ İlker

Premium hissi yüksek, mobil öncelikli, sinematik bir dijital aşk hikayesi.

## Hızlı Açılış

GitHub Pages veya doğrudan dosya açma için:

```text
index.html
```

Next.js sürümü için:

```bash
npm install
npm run dev
```

Ardından `http://localhost:3000` adresini açın.

## Neler Var?

- Sinematik tam ekran açılış
- Bölüm navigasyonu
- Mesajla başlayan hikaye akışı
- Roman sayfası hissinde tanışma bölümü
- 10 fotoğraf alanlı lightbox galeri
- 10 video alanlı Netflix tarzı arşiv
- Spotify premium hissinde müzik bölümü
- Zaman tüneli
- Bize ait sözler sahnesi
- Sayaç animasyonları
- Daktilo hissinde mektup
- Yıldızlı final sahnesi

## İçerik Değiştirme

Next.js içerikleri `data/story.ts` dosyasından değiştirilebilir.

Kişisel fotoğraflar veya video kapakları eklemek için görselleri `public` klasörüne koyup adresleri `/dosya-adi.jpg` formatında yazabilirsiniz.

## Dosya Yapısı

- `index.html`: GitHub Pages ve direkt tarayıcı açılışı için tek dosyalık sürüm
- `app/`: Next.js sayfa ve global stil yapısı
- `components/`: Sinematik deneyimin ana React bileşeni
- `data/`: Hikaye, fotoğraf, video, şarkı, zaman tüneli ve istatistik içerikleri
- `package.json`: Next.js proje komutları ve bağımlılıkları
