# Bahar ❤️ İlker

Mobil öncelikli, sinematik ve interaktif bir dijital aşk hikayesi sitesi.

## Çalıştırma

```bash
npm install
npm run dev
```

Ardından `http://localhost:3000` adresini açın.

## İçerik Değiştirme

Fotoğraf, video, şarkı, zaman tüneli ve istatistik alanları `data/story.ts` dosyasından kolayca değiştirilebilir.

Kişisel fotoğraflar veya video kapakları eklemek için görselleri `public` klasörüne koyup ilgili adresleri `/dosya-adi.jpg` formatında yazabilirsiniz.
